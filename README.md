## ✨ Overview

The **JioMart Clone** is a front-end web development project that faithfully replicates the look, feel, and layout of the **JioMart** e-commerce platform. This project is built using **pure HTML5 and CSS3 only — zero JavaScript**. It demonstrates proficiency in **semantic HTML5 markup**, **advanced CSS3 Flexbox layouts**, and **responsive design principles**.

It includes a multi-page structure — homepage, everyday essentials page, and a coupon/offers page — each populated with real product-style UI components, built entirely without any scripting language.

## 🚀 Features

- ✅ **Pixel-perfect JioMart UI** — Faithfully recreates the JioMart homepage design
- ✅ **Multi-page Navigation** — Seamless links between Home, Essentials, and Offers pages
- ✅ **Top Navigation Bar** — Logo, search bar, cart icon, and sign-in button
- ✅ **Category Carousel** — Horizontal scrollable category tiles with left/right arrows
- ✅ **Product Cards** — Wishlist icon, product name, price, MRP strikethrough & discount %
- ✅ **Promotional Banners** — Full-width and 3-column promotional image banners
- ✅ **Section Headers** — "Fashion Finds", "Festive Bestsellers", "Electronics Best Sellers", "Festive Cooking Picks"
- ✅ **Subcategory Grid** — 6-column icon grids for navigating product subcategories
- ✅ **Footer** — Multi-column footer with category links, customer account, help & support, contact info, and app download badges
- ✅ **Cloudinary CDN Assets** — High-quality images served via cloud CDN for optimal performance

## ☁️ Deployment (Vercel)

This project is fully ready for **zero-config deployment** on [Vercel](https://vercel.com). Since it's a pure static HTML/CSS site, no build step is needed.

### 🚀 One-Click Deploy

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/priyabratasahoo780/WEBSITE-1)

### 🛠️ Manual Deployment Steps

1. Go to [vercel.com](https://vercel.com) and **Sign In** with GitHub
2. Click **"Add New Project"**
3. Select your repository → `priyabratasahoo780/WEBSITE-1`
4. **Framework Preset** → Select `Other` (plain static site)
5. **Root Directory** → Leave as default (root `/`)
6. **Build & Output Settings** → Leave all blank (no build needed)
7. Click **"Deploy"** ✅

> Vercel will auto-detect `index.html` as the entry point and serve all pages instantly.

### 📄 `vercel.json` (already included)

```json
{
  "version": 2,
  "builds": [{ "src": "*.html", "use": "@vercel/static" }],
  "routes": [
    { "src": "/", "dest": "/index.html" },
    { "src": "/everyday-essentials", "dest": "/JIOMART_CLONE02.html" },
    { "src": "/happy-hour", "dest": "/JIOMART_CLONE03.html" },
    { "src": "/(.*)", "dest": "/$1" }
  ]
}
```

### 🌐 Live URL Format

After deploying, your pages will be live at:

| Page | URL |
|------|-----|
| 🏠 Home | `https://your-project.vercel.app/` |
| 🛒 Everyday Essentials | `https://your-project.vercel.app/JIOMART_CLONE02.html` |
| 🎟️ Happy Hour Coupons | `https://your-project.vercel.app/JIOMART_CLONE03.html` |
